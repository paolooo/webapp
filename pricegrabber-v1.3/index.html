Installation

Add the following libraries before </head> tag.

<link rel="stylesheet" href="bootstrap.css" />
<link rel="stylesheet" href="pricegrabber.css" />
 
<script type="text/javascript" src="jquery.min.js" ></script>
<script type="text/javascript" src="mustache.min.js" ></script>
<script type="text/javascript" src="pricegrabber.js" ></script>

Add the following inside <body> tag. By inserting the script below on any location of html doc. It will generate a list of products.

<script type="text/javascript" class="pricegrabber panel panel-default" id="pg-dcc-2000">
  $('#pg-dcc-2000').PriceGrabber({
    search: 'Cuisinart DCC-2000 Coffee Maker' // search keywords
  });
<script type="text/javascript" class="pricegrabber panel panel-default" id="pg2">
  $('#pg2').PriceGrabber({
    search: '&masterid=9273146' // search via masterid
  });
</script> </script>

Sample HTML

<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="content-type" content="text/html; charset=utf-8">
     
    <link rel="stylesheet" href="styles/bootstrap.css"/>
    <link rel="stylesheet" href="styles/pricegrabber.css"/>
     
    <script src="scripts/jquery.min.js"></script>
    <script src="scripts/mustache.min.js"></script>
    <script src="scripts/pricegrabber.js"></script>
  </head>
     
  <body>
     
    <div class="container">
      <script type="text/javascript" class="pricegrabber panel panel-default col-xs-6" id="result-1">
        $('#result-1').PriceGrabber({
          search: 'Cuisinart DCC 2650&topcat_id=27&page_id=524'
        });
      </script>
     
    </div>
     
  </body>
</html>
